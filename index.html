Alphabet, Wort, Sprache

Alphabet

  • Menge an Zeichen/Symbolen/Buchstaben
  • Endlich
  • Nicht Leer
  • Σ = { a, b }
  • Σ = { 0, 1 }
  • Σ = { a, ..., z, A, ..., Z }
  • Σ = { 1 }

Wort

  • Endliche Folge von Zeichen aus einem Alphabet Σ
  • Leeres Wort: ε
  • Worte über { 0, 1 }: ε, 0, 100, 000
  • Worte über { a, ..., z }: ε, a, ab, hello, world
Σ* Menge aller Wörter über ein Alphabet Σ

Operationen/Eigenschaften

  • Konkatenation: u + v = uv
  • Länge: | aaa | = 3; | ε | = 0
  • Präfix: Wenn u + v = w -> u ist Präfix von w
  • Suffix: Wenn u + v = w -> v ist Suffix von w
  • Teilwort: Wenn u + t + v = w -> t ist Teilwort von w
  • w ist Präfix, Suffix und Teilwort von w
  • ε ist Präfix, Suffix und Teilwort von w

Sprache

L ⊆ Σ*
Σ = { a, b }
Σ* = { ε, a, b, aa, ab, ba, ...}
  • L1 = { aa, ab, ba, bb }
  • L2 = { ε, a, aa, ab, aaa, ...}
  • L3 = { }
  • L4 = Menge aller Wörter mit keinen 2 a nebeneinander
  • L5 = Menge aller Wörter mit mehr a als b

Turingmaschine


Entwickelt 1936/37 von Alan Turing

Funktionsweise

Überführungsfunktion

  • Liest ein Symbol aus dem Bandalphabet ein
  • Kennt den aktuellen Zustand (kein Endzustand)
  • Entscheidet das Symbol mit einem neuem zu überschreiben
  • Entscheidet den Kopf nach rechts, links oder garnicht zu bewegen

Formale Bestandteile

  • Eingabealphabet; Σ = { a, b }
  • Bandalphabet, Obermenge des Eingabealphabets;
    Γ = { a, b, A, # }; enthält ein "Blank"
  • Zustände (endliche Menge an Zuständen) Q
  • Startzustand (Element von Q)
  • (akzeptierende oder nicht akzeptierende) Endzustände (Teilmenge von Q)
  • Überführungsfunktion

Entscheidbarkeit

  • Berechenbar
  • Entscheidbar
  • Semi-entscheidbar

Halteproblem

Es gibt keinen Algorithmus, der für ein beliebiges Programm entscheiden kann, ob es terminiert.
-> semi-entscheidbar

Turing-Vollständigkeit

Ein System kann alles berechnen, was eine Turingmaschine berechnen kann

Beispiele


input: 'ababbbaba'
blank: '#'
start state: q0
table:
  q0:
    b: {R: q0}
    a: {R: q1}
    '#': {R: accept}
  q1:
    a: {R: reject}
    b: {R: q0}
    '#': {R: accept}
  accept:
  reject:
					

input: 'abababbaaba'
blank: '#'
start state: q0
table:
  q0:
    a: {write: _, L: q1}
    [b, _]: {R: q0}
    '#': {R: reject}
  q1:
    b: {write: _, R: q0}
    _: {L: q1}
    '#': {R: q2}
  q2:
    [a, _]: {R: q2}
    b: {write: _, L: q3}
    '#': {R: accept}
  q3:
    [a, b, _]: {L: q3}
    '#': {R: q0}
  accept:
  reject:
